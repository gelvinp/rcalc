# Renderer commands
- Queer (Rainbow!)
- Help (Ahhh!)
- Options (screaming)

# Operators
- quot (Integer quotient)
- rem (floating point remainder)
- mod (Modulus)
- sin, cos, tan, asin, acos, atan, sinh, cosh, tanh, asinh, acosh, atanh
- min (1 param, then pop that many off the stack, and push the minimum of those)
- max (Similar but for max)
- avg (Similar but take the average)
- log (two params, log base value)
- ln (log base e value)
- exp (pow e value)
- ceil, floor, trunc, round
- sign (turn into -1, 0, or 1)
- fact (factorial)

# Vector operators
- dot
- cross
- len
- lensqr
- add, (scalar)mul, (scalar)div, sub, neg, min, max, avg

# Matrix operators
- add, sub, (scalar) mul, (scalar)div, tpose, (vec)mul, (mat)mul
- det (determinate)
- transmat (Vec2 -> mat2 translation matrix, vec3 vec4 same)
- scalemat (Vec2 -> mat2 scale matrix, vec3 vec4 same)
- rotmat (scalar -> mat2 rotation matrix, vec3 -> mat3 rotation matrix)
